# API文档管理

## 安装依赖
```
npm install
```

### 启动开发环境服务
```
npm run dev
```

### 启动线上环境服务
```
npm run prod
```

### 启动后访问
http://local.zyplayer.com

### 编译打包
```
npm run build
```
